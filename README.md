# VelvetDate - Premium Companion Rental Website

## Overview
This is a stunning, premium multi-page website template for a "Rent a Companion" service, built with modern HTML5 and Tailwind CSS.

## Features
- **Modern & Premium Design**: Dark mode aesthetic with deep purples, pinks, and gold accents.
- **Glassmorphism**: Trendy semi-transparent glass effects on cards and navigation.
- **Responsive**: Fully responsive design for mobile, tablet, and desktop.
- **Pages**:
  - `index.html`: Landing page with hero section, featured profiles, and "How it Works".
  - `browse.html`: Listing page with sidebar filters and profile grid.
  - `profile.html`: Detailed profile page with photo gallery, bio, reviews, and booking widget.

## How to Run
Simply open `index.html` in any modern web browser. No installation or server required.

## Customization
- **Images**: Images are currently pulled from Unsplash. You can replace the `src` attributes with your own local images.
- **Colors**: The color palette is defined in the `<script>` tag in the `<head>` of each file.
